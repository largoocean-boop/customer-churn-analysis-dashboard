# Customer Churn Analysis Dashboard

## Overview

This Tableau dashboard analyzes customer retention and churn behavior across 1,200 customer records. The project was designed to identify factors influencing customer attrition and provide actionable insights for improving customer retention.

## KPIs

* Total Customers: 1,200
* Churned Customers: 506
* Churn Rate: 42.17%
* Average Monthly Charges: $78.15

## Dashboard Features

* Customer Churn by Contract Type
* Customer Churn by Tenure Group
* Customer Churn by Payment Method
* Monthly Charges by Churn Status
* Interactive Contract Type Filter
* Business Insights Summary

## Key Insights

* Month-to-Month contracts experience the highest churn.
* Customers with 0–12 months tenure are most likely to leave.
* Electronic Check customers show the highest churn volume.
* Churned customers have slightly higher monthly charges than retained customers.
